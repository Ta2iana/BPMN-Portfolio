Компания «GreenTech» решила автоматизировать процесс обработки заявок на услуги. Заявки поступают от клиентов через веб-форму, где клиент указывает свои данные, услугу и дату. 
Заявки проверяются сотрудниками компании, которые проводят предварительное согласование с клиентом, создают предложение, согласуют его с руководством и затем отправляют клиенту окончательное предложение.
Процесс обработки заявок до автоматизации был следующим:
  1. Клиент заполняет заявку на сайте компании 
  2. Сотрудник проверяет заявку и создает предварительное предложение 
  3. Согласование с клиентом 
  4. Согласование с руководством компании 
  5. Отправка окончательного предложения клиенту 
  6. Завершение процесса
Задача: вам, как бизнес-аналитикам, необходимо смоделировать данный процесс с использованием BPMN, чтобы разработать систему, которая будет автоматизировать этот процесс.
